# skills

   * 在克隆时直接初始化 submodule：

     ```bash
     git clone --recurse-submodules https://github.com/Alanxtl/skills
     ```

   * 如果已经 clone 了仓库，可以手动初始化和更新 submodule：

     ```bash
     git submodule update --init --recursive
     ```
